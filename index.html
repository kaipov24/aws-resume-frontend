🌐 AWS Serverless Resume Website & CI/CD Automation (this site)

This website demonstrates my full DevOps workflow — combining a static frontend, automated infrastructure, and CI/CD. It runs entirely on AWS with zero servers to manage.

Implementation Overview

  • Built a responsive HTML/CSS/JS resume site hosted on Amazon S3 with global HTTPS delivery via CloudFront.
  • Implemented a visitor counter powered by API Gateway + Lambda (Python) + DynamoDB.
  • Wrote backend logic in Python using boto3 and secured with CORS.
  • Defined and deployed infrastructure using Terraform for full reproducibility.
  • Automated deployments with GitHub Actions:
    • Frontend workflow: syncs S3 bucket + invalidates CloudFront cache on every push
    • Backend workflow: tests Python code, validates Terraform, and deploys AWS resources automatically

Project Architecture

The entire project is built using a serverless AWS architecture, ensuring high availability, low cost, and zero maintenance overhead. Each service plays a key role in delivering this website.
